### Source code
Analysis shiny application for the manuscript, "Clonal haematopoiesis hotspot variants in patients with advanced solid tumors: analysis of a Japanese real-world clinicogenomic database", is downloadable as "app.R". Closed datasets are not uploaded, so please check the validity of the analysis with the analysis website above.
